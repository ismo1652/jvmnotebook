== Project Overview ==     

Date: Mon Apr 06 13:56:55 EDT 2009

The following entry describes a practical approach for using Clojure.  This entry 
describes how to build a simple test helper GUI application that launches a
script to compile your tests, run your tests and also runs Java's hprof and verbose garbage
collection statistics.
